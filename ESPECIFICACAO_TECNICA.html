CriArte v2 — Plataforma de Gestão de Projetos Culturais
Versão: 0.1.0
Data: Junho de 2026
Tecnologia: Next.js 15 + React 19 + TypeScript + Firebase

1. Visão Geral

CriArte v2 é uma plataforma web multi-município para gestão de projetos culturais. Permite cadastro de proponentes, submissão de projetos, avaliação e administração de editais culturais.

6.3 Administração

  • Configuração de Cidades: Cadastro de municípios, upload de logo, definição de tipos de projetos
  • Edição de Projetos: CRUD de tipos de projetos, drag-and-drop de campos entre seções, configuração de campos extras (Eixo, Módulo, Categoria)
  • Configuração de Avaliação: Critérios de avaliação por nota (0-10)
  • Atribuição de Avaliadores: Designar revisores a projetos com status "enviado"
  • Gestão de Usuários: Visualização e controle de permissões
  • Logs: Auditoria completa de ações
  • PDF: Geração de resumo do projeto em PDF (print-to-PDF)

    10. Fluxogramas

    10.1 Submissão de Projeto

    1. Usuário seleciona tipo de projeto configurado pela cidade
    2. Sistema carrega campos dinâmicos (configuração por cidade)
    3. Usuário preenche Informações Gerais → Ficha Técnica → Orçamento → Documentos → Proponente
    4. Validação em tempo real de campos obrigatórios
    5. Upload de documentos com compressão automática se > limite
    6. Salvamento automático em rascunho (Firestore)
    7. Ao enviar: status = "enviado", notificação por e-mail
    8. Admin designa avaliador → Revisor avalia (notas 0-10) → Status = "avaliado"

    10.2 Upload de Arquivos

    1. Seleção (drag-and-drop ou click)
    2. Validação de tipo (whitelist: PNG/JPG/WEBP/PDF)
    3. Validação de tamanho (máx. 10MB)
    4. Se imagem > limite: canvas resize para 1920px + compressão
    5. Se PDF > limite: compressão via pdf-lib
    6. Upload para Firebase Storage → URL salva no projeto
    7. Log de upload gerado (sucesso/falha)
